# CRO SPECIFICATION — CONVERSION FUNNELS

**GOAL:** Turn anonymous visitors into qualified leads for Iulian Bozdoghina.

---

## 1. PRIMARY CONVERSION FUNNEL
1. **Entry:** Hero CTA or Service Page CTA.
2. **Action:** "Book Free Gap Assessment" triggers multi-step qualification form.
3. **Qualification:**
   - Industry (Auto / Defence / Other)
   - Current Certification (None / Partial / Failed)
   - Revenue Bracket (<€5M / €5M-20M / €20M+)
4. **Post-Action:** Direct calendar booking (Calendly integration) + "What to prepare" email.

---

## 2. INTERACTIVE TOOL LOGIC: TISAX READINESS QUIZ

**Format:** 20 Yes/No/Partial questions.
**Scoring:**
- Yes: 5 pts
- Partial: 2 pts
- No: 0 pts

**Result Ranges:**
- **0-40 (Critical):** Immediate expert intervention needed.
- **41-75 (Warning):** Significant documentation gaps.
- **76-100 (Operational):** Ready for mock audit.

**Lead Magnet:** Score is gated. "Submit email to view your detailed readiness report and timeline estimate."

---

## 3. INTERACTIVE TOOL LOGIC: COST CALCULATOR

**Inputs:**
- **Company Size:** Slider (1-500+ employees)
- **Sites:** Number of physical locations (1-5)
- **Target Framework:** Multi-select (TISAX, CMMC, ISO 27001)

**Output:**
"Estimated Implementation Investment: **€15k - €45k**"
*Subtext: This is a strategic estimate. Book a call for a firm proposal.*
